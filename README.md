Cloud environments such as AWS, Azure, and Google Cloud provide scalability and flexibility for modern applications. However, misconfigured cloud resources, security blind spots, and uncontrolled cloud spending have become major challenges for organizations.

This project proposes a GenAI-Powered Cloud Security Copilot that automatically analyzes cloud infrastructure configurations, detects security risks, prioritizes vulnerabilities, and identifies opportunities to reduce unnecessary cloud costs.

The system uses Generative AI and risk scoring techniques to provide actionable insights through a simple dashboard, helping security and cloud operations teams quickly understand and resolve issues.
